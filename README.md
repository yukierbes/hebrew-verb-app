# Hebrew Verb Trainer

A Streamlit web app for practicing Biblical Hebrew verb identification and construction.

## Features

- Identification Quiz & Practice
- Construction Quiz & Practice
- Filter verbs by verbal root, binyan, stem, person, gender, and number

## Dataset

The application uses a curated dataset of Hebrew verb forms stored in an Excel file.

## Run Locally

1. Install requirements
2. Run the Streamlit app

streamlit run app.py

## Technology

- Python
- Streamlit
- Pandas
